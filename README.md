# Yasmeen Muhammed — Portfolio (React + Vite)

Standalone Vite version of the portfolio.

## Stack
- React 18 + Vite 7
- Tailwind CSS v4 (via @tailwindcss/vite, configured in src/index.css)
- Framer Motion, Lucide icons
- shadcn/ui components (Radix primitives)

## Structure
- `src/App.tsx` — page composition
- `src/components/sections.tsx` — Hero, About, Projects, Services, Contact, Footer
- `src/components/navbar.tsx`, `cursor.tsx`
- `src/index.css` — design tokens + Tailwind theme
